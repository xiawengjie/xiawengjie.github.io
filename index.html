初识仲景用蜜煎导剂救人！

发表于 2020-01-08 分类于 中医故事

初识仲景用蜜煎导剂救人！

“怎样才能找到张仲景呢？”一个老大的难题摆在陈皮的面前。光这么漫无目的的找，效率肯定低下，可能找上个一年半载的都未必能找到！

“贴个寻人启事？”“不行，这样也太唐突，太夸张了。”陈皮努力的回忆与张仲景生平相关的事情。忽然，他想起来张仲景有一位师傅，叫做张祖伯。如果自己能够找到张祖伯，就一定可以在那里守株待张，认识张仲景。

平常浏览器果然没有白上，百度百科没少看，幸亏还能想起来这么个人。说干就干，陈皮在老乡的指导下，走向了当地名医张祖伯的医馆！

“果然名医就是不一样，这个医馆看起来真是古乡古色，够大，够气派”，陈皮忘记了自己是穿越过来的，站在医馆面前的他发出了此般言论，引得旁边来看病的老乡用诧异的眼光看着他。

他没有理会这种目光，自顾自的走进院中，见庭院中有一亩地，一口井，二棵梅树，旁边一大块空地还上面晒着一大堆的中医材。一位50岁左右的中年人士正坐堂上，一边口中喃喃有语，一边搭脉于一位病人。

旁边坐着一位年轻人，手握着一支毛笔，正襟危坐的为老师抄写着方剂。二十七八岁的年轻，面貌棱角分明，眼神充满着坚毅，看着就让人感觉很踏实安稳，陈皮觉得此人可能就是年轻时的张仲景。

正当陈皮闻着药香，沉浸在这个氛围中的时候。突然一阵“哎哟”声打断了陈皮，只见一群人抬着一个人匆匆忙忙的从外面跑进医馆。

“救命呀，张医生，我父亲发高烧，大便已经四天没来了”一个年轻人喊道，张伯祖连忙起身赶过去查看。

他发现这个病人脉象洪大，高烧不退，腹部胀满，唇焦口燥，大便多日未通，整个人躺在地上奄奄一息，精神萎靡不振。便认为是“热伤津虚，体虚便秘”所致，需要用泻药帮助病人泻下大便，但是由于病人体质极虚，用强烈的泻药又会导致病人受不了。

张伯祖沉思半响，一时间竟没有了主张。

那个年轻人站在一旁，看老师沉思，自己便也开动脑筋。忽然，他眉间展现出一股刚毅的自信，便对老师说：“师傅，徒弟我有一个办法，可以一试”

他取来黄橙橙的蜂蜜，放进一只铜碗，就着小火煎熬，并不断地用竹筷搅动，渐渐地把蜂蜜熬成粘稠的团块。待其稍冷，便把它捏成—头稍尖的细条形状，大约有手指那么大，长2寸左右。然后将尖头朝前轻轻地塞进病人的肛门。一会儿，病人拉出一大堆腥臭的粪便，病情顿时好了一大半。

众人欢呼，张祖伯也用赞赏的眼神看着这个年轻人。

站在一旁的陈皮，更加确认这个年轻人就是张仲景，他刚刚所使用的方剂，就是他以后会记录在伤寒杂病论中的蜜煎导。

他努力的回忆蜜煎导条文：233、阳明病，自汗出。若发汗，小便自利者，虽硬不可攻之。当须自欲大便，宜蜜煎导而通之。

陈皮内心狂喜，他努力的克制着自己的欣喜，脸上依然平静淡定，思考着怎样才能和面前的这个年轻人交上朋友……..

经方医案一

发表于 2020-01-19

经方医案

今天看一个医案，脉证不符，但是用方依然有效果。

案：

48岁妇女，曾患急性肝炎，此后五年来，右肩酸痛，右手麻木，浮肿一向不治，过劳淋巴结立即肿大。又胸中苦于胀满，裤带一勒紧既感恶心。体格、营养状态一般，面色尚可，脉弱，血压正常。心下紧张如板状，有剧烈压痛，右季肋下痛尤为明显。肩酸痛严重时有短气。

我是这样想的，首先看证候群，分为了好几个部分，第一部分为右手酸痛麻木，第二部分为浮肿，第三部分为胸胁部胀满，第四部分为心下紧张如硬板状，有剧烈压痛，第五部分为脉弱。

首先，看胸胁部胀满，学过经方的人都很容易判断这个是柴胡证，首先考虑柴胡类的方子，小柴胡汤、大柴胡汤、柴胡桂枝汤、柴胡桂枝干姜汤、柴胡加龙骨牡蛎汤选一个。

如果将一部分与三部分并在一起看，是不是很像太阳少阳合病，“伤寒六七日，发热，微恶寒，支节烦疼，微呕，心下支结，外证未去者，柴胡桂枝汤主之。”

但是此处脉弱，如果有表证，脉应该是浮脉，且应恶寒，发热，这里都无，所以排除柴胡桂枝汤。

现在，我们来看第四部分，心下紧张如硬板状。“按之不痛者为虚，痛者为实”，此处内有实证？如果内有实证，就可以选用大柴胡汤！

但是，这里脉弱呀，大柴胡汤内有大黄等攻下药，用在此处是否合适呢？这个就是我犹豫的点。

我们一起来看一下答案：

以上所见证与“心下急，郁郁微烦，胸胁苦满，心下痞硬，呕吐，腹满痛”之大柴胡汤条文几乎一致。由于右肩部酸痛与右手麻木、右季肋紧张压痛相关联，故胸胁苦满有时轻快，有时不轻快。尽管脉较弱，仍与大柴胡汤加葛根5g。服用本方10日，五年之肩酸痛、右手麻木、胸闷不舒，几乎痊愈。一个月之后，乘汽车、电车时晕车亦消失，心下痞硬和苦满等症状好转。服用3个月，宿疾一扫而光。

这位医家于此处用的便是大柴胡汤，上述的大柴胡汤条文处，无脉证记录，但可以推测，脉应为实脉。

此案脉弱，依然用了大柴胡汤，且一吃就吃了3个月，宿疾还一扫而光，急则治其标，缓则治其本？这一块依然不是特别理解，我存疑，望有识之士解惑~

经方故事开篇！！持续连载ing

发表于 2020-01-08 分类于 中医故事

经方故事开篇！！持续连载ing

夏陈皮是一名十分普通的年轻人，在二十几岁的年纪，与绝大部分年轻人一样，拥有扁扁的口袋与一副满腔热血的心。除了这二样东西之外，他一无所有，唯一的兴趣爱好就是研究学习中医。

上以疗君亲之疾，下以救贫贱之厄，中以保身长全，以养其生。这句记载在伤寒杂病论序言上的话，是他一直以来追求的目标。

一日，陈皮正在书房之内，潜心研究伤寒杂病论。忽然窗外狂风大作，陈皮站起身来，拉开窗户，想去感受一下这天地的阴阳激荡之气。这时，一篇树叶随风飘荡，摇曳进入了陈皮的书房，不偏不倚正好落在了未合上的那本伤寒杂病论上，盖住了勤求古训，博采众方八个大字，刹那间电闪雷鸣，一道闪电从天空击落，正巧落在陈皮的眼前，一阵眩晕，陈皮昏倒了过去。

等到陈皮醒来之时，四周是那么的陌生！原来充满现代风格的家变成了一间简陋的茅草屋，自己身上的衣服不知何时竟然变成了汉朝时的衣服。陈皮大为惊讶，自言自语道：难道这狗血的穿越剧情，落到了我的身上？

他连忙推开茅屋的门，想看看外面的世界到底发生了什么？这一看傻眼了，外面再也没有高楼大厦，没有穿流不息的汽车，只有高不过二层的木制建筑，来往的通道上零散的留着马车压过的痕迹。

他自言自语道：完了，这下真的穿越了。赶快找个人问一下现在是什么时候？我现在在哪里？

“老乡，我想问一下，现在是什么时候呀！这个地方叫什么名字呀！”他连忙拉住一位过路的中年大叔问道。“现在是东汉，你处在的地方叫做南郡涅阳县”大叔回答道。

陈皮沉思道，“南郡涅阳县不就是现代的河南南阳县嘛！”

东汉、南阳、伤寒杂病论、中医这些词同时涌上陈皮的心头，莫不是上天看我对中医极其热爱，特地送我回到汉朝时期，跟师张仲景？

陈皮边想边笑，连忙继续追问：“老乡，那你知道这周围有什么有名的医生嘛？我肚子有点痛！”

“哦！找医师是吧！你算找对地方了，我们这边医生很多，流派也很多，有桂枝科，有麻黄科，有葛根科，有柴胡科。。。。”，“你最好都去看看，有的人在一个医师那里，效果不好，换一个医师效果就非常好，常常一二付药就彻底痊愈了。”大哥耐心的回答道。

陈皮听完大哥的阐述才了解到，原来这个张仲景这个时代，有很多厉害的中医师。有的人擅长用桂枝，就形成了桂枝科，有的人擅长用麻黄，就形成了麻黄科。有的医师可能一辈子家里面就传下来那么一二个方子，就一直用那么一两个方子。用对了，效果非凡，用错了，就没有丝毫效果。

这时，陈皮才了解到，张仲景是这个时代的极大成者。他在这个时代勤求古训，博采众方，充分吸纳了当时各个医家流派的长处，最终才写出了流传千古的伤寒杂病论，确定了辨证论治的准则！

想到这，陈皮陷入了沉思，愈发的期待能见到仲景本人……….

陈皮装病骗仲景

发表于 2020-01-08 分类于 中医故事

陈皮这时候还在想着：怎样才能与仲景搭上话？又不会显得太尴尬呢？毕竟以前只有搭讪女孩子的经验，又没有主动搭讪过一个男的，毫无经验呀！

陈皮左思右想，既然这是医馆，那我就先装个病人吧！找仲景给我看病，完事儿过二天病好了，我再回来买点礼物来感谢他。然后说自己也立志于方术，想跟他交个朋友，一起学习！

行！就这么办！陈皮一边思考着，一边走到张仲景的面前说：“你好，这位医师，我想找你看个病！”，张仲景答到：“你好，请问你哪里不舒服呢？”陈皮在脑海里面疯狂的搜索条文，要想一个安全的方子来用一下。

135、伤寒六七日，结胸热实，脉沉而紧，按之石硬者，大陷胸汤主之。这个不行，我主述这个，万一他一摸脉，一腹诊没有这个情况怎么办？岂不是露馅了？

金匮、胸痹，心中痞气，气结在胸，胸满，胁下逆抢心，枳实薤白桂枝汤主之，人参汤亦主之。我引导他用这里的人参汤吧！反正我脾胃素来虚寒，喝个一两剂对我也有好处。

“医生，我感觉肚子里凉飕飕的，有一点痛”陈皮答到，仲景继续追问“那你最近吃了些什么呢？”，“最近正是鸭子肥美的时候，一时贪口，连续吃了四天鸭子了。”，“来，把你的手伸出来，我给你摸一下脉”

陈皮伸出他的手，放到桌子，仲景一边搭脉一边续问道：“平素的时候，胃口怎么样，大便有没有成型？”，“胃口还好，就是大便经常不成型”，“嗯，好了，我知道了”

张仲景挪开手，拿起毛笔在纸上写下：人参、甘草、干姜、白术各三两，以水八升，煮取三升，温服一升，日三服。

“好了，你拿着这个处方去拿药吧！”张仲景说，陈皮为了跟仲景多熟悉一会，特地问道：“医生呀，我这是个什么毛病呀，怎么会突然这样呢？”，仲景解释道：“你呀，平素就中焦虚寒，这一次又过量的食用鸭子。你看那个鸭子常年的在水里游呀游，属于性寒之品，大量的服用就会使你寒上加寒，故而此次腹部冷痛”

“我这次给你开的方剂是为人参汤，服用之后应该就好了。好了之后，你下次再来，我给你改方，常服用，可渐渐的改变你中焦虚寒的体质”。

“好的，谢谢您，哪我下次再来”，陈皮一边道谢，一边暗喜，心中期待着下次再来………..
